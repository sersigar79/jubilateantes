# Calculadora de Jubilación Anticipada (FIRE + Lotería + Pensiones)

Una herramienta sencilla, visual y muy potente para calcular:

- A qué edad podrías jubilarte.
- Cuánto dinero necesitas para vivir sin trabajar.
- Cuánto deberías ahorrar cada año.
- Cuánto te tendría que tocar en la lotería para jubilarte antes.
- Cómo afectan los ingresos pasivos (alquileres, etc.).
- Cómo influye una pensión pública estimada.

La web está diseñada para ser **fácil de usar**, **atractiva**, **rápida** y **entendible para todo el mundo**.

---

## 🚀 Características

- Cálculo del objetivo FIRE (regla del 4%).
- Proyección de tu capital año a año.
- Gráfica comparativa entre tu dinero y el objetivo necesario.
- Escenarios con lotería, ingresos pasivos y pensión pública.
- Interfaz clara y moderna.
- Funciona en cualquier país e idioma (el texto es fácilmente traducible).
- 100% orientativa, sin asesoramiento financiero.
